What is Time Complexity?

Amount of time taken as the function of input size.

1 - n ( number of operations = n);

- Consider Worst case scenerio to get the time complexity

Notations to measure Time Complexity.

Big O -> Worst Case
Theta -> Avg.
Omega -> Best Case

Types

# Constant Time or O(1);

- Any Task gets completed in single operation.
- If number of operations are fixed. e.g. if n = 10; iterating over that is constant tc.

# Linear TC or O(n)

- n is not given so n number of operation will get performed.

# Quadratic TC or O(n^2);

- two nested loops running n times each.

# O(log n)

- number of operations drastically reduces than the expected operations in worst case.
